## Design decisions

1. **Pinned major versions, not exact**: Dependencies use major-version specifiers (e.g., `git2 = "0.19"`, `notify = "7"`) to allow patch updates while preventing breaking changes.

2. **git2 0.19 over 0.20**: Used 0.19 since it is the version cargo resolved as latest compatible. The 0.20 series was available but 0.19 is sufficient for the diff/status/blame operations needed.

3. **notify v7 over v8**: The ticket specified "use version 7+"; v7 is well-tested and v8 was available but not required.

4. **Added links and status command modules now**: The IPC contracts define `get_backlinks` and `workspace_status` commands that need command wrapper modules. Adding the stubs now avoids drift between the IPC contracts and the module structure.
