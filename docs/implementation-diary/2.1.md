# Task 2.1: App Shell and Zen Title Bar Baseline

**Date**: 2026-03-11
**Status**: DONE
**Branch**: epic0-claude

## What was done

Created the foundational layout components for the Vigil desktop app shell using Svelte 5 runes and Tailwind v4 design tokens.

### Components created

| Component | File | Purpose |
|---|---|---|
| `TitleBar.svelte` | `src/lib/components/layout/TitleBar.svelte` | Minimal 32px custom title bar with `data-tauri-drag-region` for window dragging, "Vigil" label, and window control buttons (minimize, maximize, close) using dynamic Tauri API imports |
| `AppShell.svelte` | `src/lib/components/layout/AppShell.svelte` | Top-level flex-column container arranging TitleBar (top), main content (middle), StatusBar (bottom) via Svelte 5 snippets |
| `WorkspaceGrid.svelte` | `src/lib/components/layout/WorkspaceGrid.svelte` | Horizontal flex layout with optional slots for ActivityRail, Sidebar, main editor area, and right panel |

### Files modified

| File | Change |
|---|---|
| `src/lib/components/layout/index.ts` | Updated barrel file to export AppShell, TitleBar, WorkspaceGrid |
| `src/routes/+page.svelte` | Replaced placeholder with full app shell composition: AppShell wrapping TitleBar, WorkspaceGrid with empty-state message, and a minimal status bar footer |

### Design decisions

- **Dynamic Tauri imports**: TitleBar uses `await import('@tauri-apps/api/window')` inside try/catch so the component works in both Tauri and browser-preview contexts without build errors.
- **Snippet-based composition**: AppShell accepts `titlebar`, `children`, and `statusbar` snippets. WorkspaceGrid accepts `activityRail`, `sidebar`, `children`, and `rightPanel` snippets. This lets parent pages inject zone content without tight coupling.
- **Status bar placeholder**: A minimal `<footer>` in +page.svelte acts as the status bar slot until the real StatusBar component is built in a later ticket.
- **Tailwind token usage**: All colors reference the design tokens from app.css (bg-surface-base, text-text-primary, text-text-muted, border-surface-border, etc.).
