**Steps:**
1. Install `monaco-editor` npm package.
2. Create `src/lib/features/editor/monaco-config.ts` with Vigil dark theme, performance options, and language detection.
3. Rewrite `src/lib/features/editor/CodeEditor.svelte` to lazy-load and mount Monaco, syncing with `codeStore`.

## 2026-03-12T00:01 -- Implementation Complete

**Files created/modified:**
1. `package.json` -- added `monaco-editor` dependency (installed via npm).
2. `src/lib/features/editor/monaco-config.ts` -- NEW. Vigil dark theme definition, performance-focused editor options, language detection map, lazy Monaco loader.
3. `src/lib/features/editor/CodeEditor.svelte` -- REWRITTEN. Replaced textarea skeleton with lazy-loaded Monaco editor. Mounts on first render, syncs content changes to codeStore, handles file switches by updating model language and content.
4. `src/lib/features/editor/index.ts` -- UPDATED. Added barrel exports for monaco-config.

**Key implementation details:**
- Monaco is loaded via dynamic `import('monaco-editor')` on first CodeEditor mount.
- `automaticLayout: true` handles container resizing automatically.
- Minimap disabled, smooth scrolling off, occurrence highlighting off for performance.
- `glyphMargin: true` preserved for future git gutter decorations (Task 3.6).
- Vigil dark theme uses exact color values from `app.css` design tokens.
- Language detection covers 40+ file extensions mapping to Monaco language IDs.
