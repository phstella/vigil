**Blocker:** The Rust toolchain (`cargo`, `rustc`, `rustfmt`, `clippy`) is not installed on this machine. The `cargo test`, `cargo clippy`, and `cargo fmt --check` commands cannot be executed locally.

**Mitigation:** These checks are enforced in CI (`.github/workflows/ci.yml`) which installs the Rust toolchain via `dtolnay/rust-toolchain@stable`. The existing CI runs:
- `cargo fmt --manifest-path src-tauri/Cargo.toml --all --check`
- `cargo clippy --manifest-path src-tauri/Cargo.toml --workspace --all-targets -- -D warnings`
- `cargo test --manifest-path src-tauri/Cargo.toml --workspace`

All of these passed on the most recent merge to master (commit d05bd4f).
