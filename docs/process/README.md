# Process Docs

Repository-level delivery standards for human and local agent workflows.

## Files

1. `epic-completion-checklist.md` -- required checklist for Epic/task closeout.
2. `agent-handoff-template.md` -- required structure for handoff/review context docs.
3. `pr-review-rubric.md` -- severity-first rubric for implementation sufficiency reviews.
4. `epic-3-completion-checklist-2026-03-12.md` -- prefilled Epic 3 baseline checklist from latest deep re-review.
5. `product-owner-orchestrator-agent-prompt.md` -- reusable PO-style orchestration prompt for delegating work to developer agents.
6. `product-owner-epic3-first-run-message.md` -- copy/paste kickoff message to launch Epic 3 recovery orchestration.

## Policy

1. Canonical scope comes from `docs/tickets/00-unified-vigil-backlog.md`.
2. If evidence is missing, status must be `PARTIAL`, not `PASS`.
3. Runtime behavior must be verified, not inferred from compile success only.
