# Task 0.1: Scaffold SvelteKit Project

## Goal

Create the base SvelteKit project that will serve as Vigil's frontend shell.

## Commands

```bash
npm create svelte@latest vigil
```

When prompted, select:
- **Template:** Skeleton project
- **Type checking:** Yes, using TypeScript syntax
- **Additional options:** Add ESLint, Add Prettier

Then install dependencies:

```bash
cd vigil
npm install
```

## Files Created

After scaffolding, the following structure should exist:

```
vigil/
├── src/
│   ├── app.html
│   ├── app.d.ts
│   └── routes/
│       └── +page.svelte
├── static/
│   └── favicon.png
├── package.json
├── svelte.config.js
├── tsconfig.json
├── vite.config.ts
├── .eslintrc.cjs
├── .prettierrc
└── .gitignore
```

## Acceptance Criteria

- [ ] `npm run dev` starts a Vite dev server on `http://localhost:5173`
- [ ] Visiting `http://localhost:5173` renders the default Svelte skeleton page
- [ ] `npm run check` passes TypeScript validation
- [ ] `npm run lint` passes ESLint checks
