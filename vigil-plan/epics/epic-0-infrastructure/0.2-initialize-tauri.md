# Task 0.2: Initialize Tauri v2 Inside the Project

## Goal

Add the Tauri v2 Rust backend shell to the existing SvelteKit project so the
app runs as a native desktop window.

## Commands

Install the Tauri CLI and JS API:

```bash
npm install -D @tauri-apps/cli
npm install @tauri-apps/api
```

Initialize the Tauri backend:

```bash
npx tauri init
```

When prompted, provide these values:

| Prompt | Value |
|---|---|
| App name | `vigil` |
| Window title | `Vigil` |
| Frontend dev URL | `http://localhost:5173` |
| Frontend dist directory | `../build` |
| Dev command | `npm run dev` |
| Build command | `npm run build` |

Add the Tauri script to `package.json`:

```json
{
  "scripts": {
    "tauri": "tauri"
  }
}
```

## Acceptance Criteria

- [ ] `npx tauri dev` compiles the Rust backend without errors
- [ ] A native window opens rendering the SvelteKit dev page
- [ ] Hot-reload works — editing a `.svelte` file updates the window instantly
- [ ] `npx tauri build` produces a native binary in `src-tauri/target/release/`
