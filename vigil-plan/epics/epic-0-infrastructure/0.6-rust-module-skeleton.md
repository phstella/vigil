# Task 0.6: Create Rust Module Skeleton

## Goal

Establish the Rust source directory structure with empty module declarations
so all future backend tasks can work in isolated files without merge conflicts.

## Files to Create

```
src-tauri/src/commands/mod.rs
src-tauri/src/commands/fs.rs
src-tauri/src/commands/git.rs
src-tauri/src/commands/search.rs
src-tauri/src/commands/workspace.rs
src-tauri/src/indexer/mod.rs
```

## Commands

```bash
mkdir -p src-tauri/src/{commands,indexer,git}
touch src-tauri/src/commands/{mod.rs,fs.rs,git.rs,search.rs,workspace.rs}
touch src-tauri/src/indexer/{mod.rs,file_watcher.rs,link_graph.rs,search_index.rs,tag_index.rs}
touch src-tauri/src/git/{mod.rs,differ.rs}
```
