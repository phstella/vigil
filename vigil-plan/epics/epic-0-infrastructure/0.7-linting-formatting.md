# Task 0.7: Configure Linting & Formatting Standards

## Goal

Enforce consistent code quality across both Rust and TypeScript with automated
linting and formatting tools.

## Commands

```bash
npm install -D prettier-plugin-svelte
```

## Files to Create

### `rustfmt.toml`

```toml
edition = "2021"
max_width = 100
tab_spaces = 4
use_field_init_shorthand = true
```

### `clippy.toml`

```toml
cognitive-complexity-threshold = 30
```

### `.prettierrc`

```json
{
  "semi": true,
  "singleQuote": true,
  "tabWidth": 2,
  "trailingComma": "none",
  "printWidth": 100,
  "plugins": ["prettier-plugin-svelte"],
  "overrides": [
    {
      "files": "*.svelte",
      "options": {
        "parser": "svelte"
      }
    }
  ]
}
```

## Files to Modify

### `src-tauri/Cargo.toml`

Add linting configuration:

```toml
[lints.clippy]
all = "warn"
pedantic = "warn"
```

### `.eslintrc.cjs`

Extend the default SvelteKit config to enforce:

```js
module.exports = {
  // ... existing extends ...
  rules: {
    '@typescript-eslint/no-unused-vars': 'error',
    'svelte/no-at-html-tags': 'warn'
  }
};
```

### `package.json` — add scripts

```json
{
  "scripts": {
    "lint": "eslint . && cd src-tauri && cargo clippy -- -D warnings",
    "format": "prettier --write . && cd src-tauri && cargo fmt",
    "format:check": "prettier --check . && cd src-tauri && cargo fmt --check",
    "check": "svelte-kit sync && svelte-check --tsconfig ./tsconfig.json"
  }
}
```

## Standards Enforced

| Language | Formatter | Linter | Config File |
|---|---|---|---|
| Rust | `cargo fmt` (rustfmt) | `cargo clippy` | `rustfmt.toml`, `clippy.toml` |
| TypeScript | Prettier | ESLint | `.prettierrc`, `.eslintrc.cjs` |
| Svelte | Prettier (plugin) | ESLint (svelte plugin) | `.prettierrc`, `.eslintrc.cjs` |
| CSS | Prettier | — | `.prettierrc` |

## Acceptance Criteria

- [ ] `npm run lint` runs both ESLint and Clippy with zero errors on the scaffold
- [ ] `npm run format` formats all `.ts`, `.svelte`, `.css`, and `.rs` files
- [ ] `npm run format:check` passes in CI (no unformatted files)
- [ ] `npm run check` passes Svelte type-checking
- [ ] Clippy `pedantic` warnings are active for all Rust code
