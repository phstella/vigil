# Task 1.1: File System Read/Write Commands

## Goal

Expose Tauri commands for reading, writing, renaming, and deleting plain-text
and Markdown files from the local filesystem.

## Files to Modify

### `src-tauri/src/commands/fs.rs`

Implement these `#[tauri::command]` functions:

```rust
#[tauri::command]
pub fn read_file(path: String) -> Result<String, String>
```
- Read file as UTF-8 string.
- Return `Err` with a human-readable message if the file doesn't exist or isn't valid UTF-8.

```rust
#[tauri::command]
pub fn write_file(path: String, content: String) -> Result<(), String>
```
- **Atomic write**: write to `{path}.tmp`, then `std::fs::rename` to the final path.
- This prevents data loss if the app crashes mid-write.
- Create parent directories if they don't exist.

```rust
#[tauri::command]
pub fn list_directory(path: String) -> Result<Vec<FileEntry>, String>
```
- Return `Vec<FileEntry>` where `FileEntry` is:
  ```rust
  #[derive(Serialize)]
  pub struct FileEntry {
      pub name: String,
      pub path: String,
      pub is_dir: bool,
      pub extension: Option<String>,
  }
  ```
- Sort: directories first, then alphabetical by name (case-insensitive).

```rust
#[tauri::command]
pub fn rename_file(old_path: String, new_path: String) -> Result<(), String>
```

```rust
#[tauri::command]
pub fn delete_file(path: String) -> Result<(), String>
```
- Use the `trash` crate to move to OS trash — **never** permanently delete.

### `src-tauri/src/main.rs`

Register all commands:

```rust
tauri::Builder::default()
    .invoke_handler(tauri::generate_handler![
        commands::fs::read_file,
        commands::fs::write_file,
        commands::fs::list_directory,
        commands::fs::rename_file,
        commands::fs::delete_file,
    ])
    .run(tauri::generate_context!())
    .expect("error while running vigil");
```

## Acceptance Criteria

- [ ] `invoke('read_file', { path: '/tmp/test.md' })` returns file contents from the browser console
- [ ] `invoke('write_file', { path: '/tmp/test.md', content: '# Hello' })` creates the file on disk
- [ ] Written file is verifiable via `cat /tmp/test.md` in terminal
- [ ] `invoke('list_directory', { path: '/tmp' })` returns an array of `FileEntry` objects
- [ ] `invoke('delete_file', { path: '/tmp/test.md' })` moves the file to OS trash (not permanently deleted)
- [ ] Reading a nonexistent file returns a descriptive error string
