# Task 2.1: Minimal Title Bar (Zen Mode)

## Goal

Replace the OS title bar with a minimal, unobtrusive Tauri-draggable bar
that stays out of the way — supporting the "zen mode by default" philosophy.

## Files to Create

### `src/lib/components/layout/TitleBar.svelte`

```svelte
<!-- Structure -->
<header data-tauri-drag-region class="titlebar">
  <div class="title-area">
    <span class="workspace-name">{workspaceName}</span>
    <span class="separator">—</span>
    <span class="file-name">{fileName}</span>
    {#if isDirty}<span class="dirty-dot">●</span>{/if}
  </div>
  <div class="window-controls">
    <button onclick={minimize}>─</button>
    <button onclick={maximize}>□</button>
    <button onclick={close} class="close-btn">✕</button>
  </div>
</header>
```

**Behavior:**
- `data-tauri-drag-region` on the header enables native window dragging.
- Window control buttons call Tauri window APIs:
  ```ts
  import { getCurrentWindow } from '@tauri-apps/api/window';
  const appWindow = getCurrentWindow();
  appWindow.minimize();
  appWindow.toggleMaximize();
  appWindow.close();
  ```
- Display the current workspace name and active file name (from stores).
- Show a dirty indicator (●) when the file has unsaved changes.

**Zen mode integration:**
- The title bar should be as minimal as possible. The mock shows no visible
  title bar — the sidebar header ("EXPLORER") sits at the very top.
- Consider making the title bar a thin (h-7) transparent overlay that only
  shows window controls on hover, or embedding window controls into the
  top-right corner of the activity bar / sidebar area.
- File name and dirty indicator can be shown in the status bar instead.

**Styling:**
```
h-7
bg-transparent
backdrop-blur-none
flex items-center justify-end
px-2
text-vigil-text-muted text-xs
select-none
```

Window control buttons:
- `w-8 h-8 flex items-center justify-center rounded hover:bg-white/10`
- Close button hover: `hover:bg-red-500/80`

## Files to Modify

### `src-tauri/tauri.conf.json`

Disable native decorations and enable transparency:

```json
{
  "app": {
    "windows": [
      {
        "title": "Vigil",
        "width": 1280,
        "height": 800,
        "minWidth": 800,
        "minHeight": 600,
        "decorations": false,
        "transparent": true
      }
    ]
  }
}
```

## Acceptance Criteria

- [ ] The app launches with no OS title bar
- [ ] The custom bar is draggable — clicking and dragging moves the window
- [ ] Minimize button minimizes the window
- [ ] Maximize button toggles between maximized and restored state
- [ ] Close button closes the app
- [ ] Workspace name and file name display correctly
- [ ] Dirty indicator appears when a file has unsaved changes
- [ ] Works on both Linux (X11/Wayland) and Windows
