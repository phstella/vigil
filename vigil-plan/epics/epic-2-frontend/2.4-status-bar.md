# Task 2.4: Status Bar

## Goal

Display knowledge-management metadata and editor context at the bottom of
the window: git branch, sync status, note/word counts, active file tags,
and version — matching the design mock.

## Files to Create

### `src/lib/components/layout/StatusBar.svelte`

**Structure:**
```svelte
<footer class="status-bar">
  <div class="left-section">
    {#if gitBranch}
      <span class="branch">
        <svg><!-- git branch icon --></svg>
        {gitBranch}
      </span>
    {/if}
    {#if syncStatus}
      <span class="sync-status">
        <span class="sync-dot" class:syncing={syncStatus === 'syncing'}></span>
        {syncLabel}
      </span>
    {/if}
  </div>
  <div class="right-section">
    <span class="note-count">Notes: {openNoteIndex} / {totalNotes}</span>
    {#if fileTags.length > 0}
      <span class="tags">Tags: {fileTags.join(', ')}</span>
    {/if}
    <span class="version">VIGIL V{appVersion}</span>
  </div>
</footer>
```

**Data sources:**

| Field | Source | Store |
|---|---|---|
| Git branch | `git_repo_status` Tauri command | `gitState.branch` |
| Sync status | File watcher activity indicator | `gitState.syncStatus` |
| Note count (current / total) | Derived from `filesState.fileTree` (count `.md` files) | `filesState` derived |
| Tags for active file | `get_file_tags` Tauri command | `editorState` / `tagsState` |
| App version | Hardcoded or from `package.json` | constant |

**Sync status indicator:**
- Green dot + "Syncing..." when the file watcher is processing events.
- Fades to idle after 2 seconds of no activity.
- This is a local indicator (file watcher active), not cloud sync.

**Behavior:**
- Tags update when switching files.
- Note count updates when files are created/deleted.
- Git branch updates when switching workspaces.
- All sections are non-interactive in the MVP (display only).

## Acceptance Criteria

- [ ] Status bar renders at the bottom of the window, full width
- [ ] Git branch icon + name displays on the left when in a git repository
- [ ] Green "Syncing..." dot pulses when file watcher events are active
- [ ] "Notes: X / Y" shows current note index and total `.md` file count
- [ ] Tags for the active file display on the right (e.g., "Tags: prothean, relic")
- [ ] Tags section hides when the active file has no tags
- [ ] "VIGIL V1.0.0" version label renders on the far right
- [ ] Status bar height is exactly `h-7` (28px), never grows
- [ ] Status bar does not scroll — always fixed at the bottom
