# Task 2.5: Root Layout Composition

## Goal

Wire all layout components into the main `+page.svelte` to form the complete
IDE-like application shell.

## Files to Modify

### `src/routes/+page.svelte`

Compose the full layout:

```svelte
<script>
  import TitleBar from '$lib/components/layout/TitleBar.svelte';
  import ActivityBar from '$lib/components/layout/ActivityBar.svelte';
  import Sidebar from '$lib/components/layout/Sidebar.svelte';
  import SplitPane from '$lib/components/layout/SplitPane.svelte';
  import StatusBar from '$lib/components/layout/StatusBar.svelte';
  import EditorRouter from '$lib/components/editor/EditorRouter.svelte';
</script>

<div class="h-screen flex flex-col bg-vigil-bg text-vigil-text overflow-hidden">
  <TitleBar />

  <div class="flex flex-1 min-h-0">
    <ActivityBar />
    <Sidebar />

    <main class="flex-1 min-w-0 overflow-hidden">
      <SplitPane>
        {#snippet left()}
          <EditorRouter pane="left" />
        {/snippet}
        {#snippet right()}
          <EditorRouter pane="right" />
        {/snippet}
      </SplitPane>
    </main>
  </div>

  <StatusBar />
</div>
```

**`EditorRouter` logic:**
Routes `.md` files to `WysiwygEditor` and all other files to `MonacoEditor`.

**Layout rules:**
- Outer container: `h-screen flex flex-col` вҖ” fills the entire window.
- Title bar: minimal height (`h-7`), nearly invisible, `shrink-0`.
- Middle area: `flex flex-1 min-h-0` вҖ” activity bar + sidebar + editor.
- Activity bar: fixed width (`w-12`), always visible, `shrink-0`.
- Sidebar: fixed width (`w-60`), collapsible/auto-hiding.
- Editor area: `flex-1 min-w-0 overflow-hidden` вҖ” fills all remaining space.
- Status bar: fixed height (`h-7`), `shrink-0`.
- `overflow-hidden` on the outer div prevents any scrollbars on the shell.
- `min-h-0` on the flex child prevents content from overflowing.

### `src/routes/+layout.svelte`

Ensure the root layout is clean:

```svelte
<script>
  import '../app.css';
  let { children } = $props();
</script>

{@render children()}
```

## Acceptance Criteria

- [ ] The app renders the complete layout: title bar, sidebar, editor area, status bar
- [ ] No scrollbars appear on the outer window shell
- [ ] The editor area fills all remaining vertical and horizontal space
- [ ] Resizing the window correctly resizes the editor area (sidebar and bars stay fixed)
- [ ] The window background is `#0d0f12` (vigil-bg) with no white flashes on resize
- [ ] All components are visible and positioned correctly at 1280x800 minimum
- [ ] The layout works at any window size down to 800x600
