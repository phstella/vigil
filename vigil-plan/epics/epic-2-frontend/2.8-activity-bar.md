# Task 2.8: Activity Bar (Icon Rail)

## Goal

Build a narrow vertical icon rail on the far left edge of the window that
switches the sidebar between different panel views: Explorer, Search,
Graph, and Tags.

## Files to Create

### `src/lib/components/layout/ActivityBar.svelte`

**Structure:**
```svelte
<nav class="activity-bar">
  <div class="top-icons">
    {#each panels as panel}
      <button
        class="activity-icon"
        class:active={activePanel === panel.id}
        onclick={() => setActivePanel(panel.id)}
        title={panel.label}
      >
        {@html panel.icon}
      </button>
    {/each}
  </div>
  <div class="bottom-icons">
    <button class="activity-icon" onclick={openSettings} title="Settings">
      {@html settingsIcon}
    </button>
  </div>
</nav>
```

**Panels:**

| Icon | Panel ID | Label | Sidebar Content |
|---|---|---|---|
| Folder tree icon | `explorer` | Explorer | File tree (existing Task 2.2) |
| Magnifying glass | `search` | Search | Workspace search input + results |
| Network/graph icon | `graph` | Graph | Note connection graph (Task 3.9) |
| Tag/label icon | `tags` | Tags | Tag browser list |

**Behavior:**
- Clicking an icon activates that panel in the sidebar.
- Clicking the already-active icon **toggles the sidebar closed** (same as `Ctrl+B`).
- The active icon gets a teal left border indicator and brighter icon color.
- The settings gear at the bottom opens a settings panel (stub for MVP — just a placeholder).
- Active panel ID is stored in the `settings` store.

**Styling:**
```
w-12
bg-vigil-bg
border-r border-vigil-border
flex flex-col justify-between
py-2
shrink-0
```

Icon buttons:
```
w-12 h-12
flex items-center justify-center
text-vigil-text-muted
hover:text-vigil-text
transition-colors
relative
```

Active state:
```
text-vigil-teal
before:absolute before:left-0 before:top-2 before:bottom-2
before:w-0.5 before:bg-vigil-teal before:rounded-r
```

**Icons:**
Use inline SVGs (no icon library). Each icon is 20x20px, stroke-based,
1.5px stroke width, matching the mock's clean line-art style.
