# Task 3.6: Bidirectional Linking UI + `[[` Autocomplete

## Goal

Make `[[wikilinks]]` in Markdown files visually styled and clickable, provide
an inline autocomplete popup when typing `[[`, and show a backlinks panel
listing all files that reference the current file.

## Files to Create/Modify

### Part 1: `[[` Autocomplete Trigger (WYSIWYG Editor)

**File:** `src/lib/components/editor/extensions/wikilink.ts`

This is a Tiptap extension (for the WYSIWYG editor from Task 3.2):

```ts
import { Extension } from '@tiptap/core';
import { Plugin, PluginKey } from '@tiptap/pm/state';

// Input rule: typing [[ triggers the autocomplete popup
// The extension watches for the [[ pattern and opens a floating menu

export const WikilinkExtension = Extension.create({
  name: 'wikilink',

  addProseMirrorPlugins() {
    return [
      new Plugin({
        key: new PluginKey('wikilink-autocomplete'),
        // Watch for [[ input, open popup with file list
        // Calls fuzzyFind() from Tauri to get matching files
        // Completing a selection inserts [[target]] as a wikilink node
      }),
    ];
  },
});
```

**Autocomplete popup behavior:**
- Typing `[[` opens a small floating popup below the cursor.
- The popup lists files matching the text after `[[` (e.g., `[[Pro` filters
  to files starting with "Pro").
- Uses `fuzzyFind()` from Task 1.5 for matching.
- `ArrowUp`/`ArrowDown` to navigate, `Enter` to select, `Escape` to dismiss.
- Selecting a file inserts `[[file-name]]` as a styled wikilink node.
- The popup shows file names with their directory path in muted text.
- Popup styling: same glassmorphism as the omnibar (`bg-vigil-surface/80
  backdrop-blur-xl border border-vigil-border rounded-lg shadow-xl`).

### Part 1b: Wikilink Styling & Click Navigation

**In the WYSIWYG editor (Tiptap):**
- Wikilinks render as a custom Tiptap node with teal color + dotted underline.
- Clicking a wikilink navigates to the linked file.
- Defined in the `wikilink.ts` extension's `addNodeView()`.

**In the Monaco editor (code files):**
- Register a Monaco decoration provider for `[[...]]` patterns:

```ts
function decorateWikilinks(editor: monaco.editor.IStandaloneCodeEditor) {
  const model = editor.getModel();
  if (!model) return;

  const content = model.getValue();
  const regex = /\[\[([^\]|]+)(?:\|[^\]]+)?\]\]/g;
  const decorations: monaco.editor.IModelDeltaDecoration[] = [];

  let match;
  while ((match = regex.exec(content)) !== null) {
    const startPos = model.getPositionAt(match.index);
    const endPos = model.getPositionAt(match.index + match[0].length);
    decorations.push({
      range: new monaco.Range(
        startPos.lineNumber, startPos.column,
        endPos.lineNumber, endPos.column
      ),
      options: {
        inlineClassName: 'wikilink-decoration',
        hoverMessage: { value: `Go to: ${match[1]}` },
      },
    });
  }

  wikilinkDecorationIds = editor.deltaDecorations(wikilinkDecorationIds, decorations);
}
```

- Ctrl+Click on a wikilink in Monaco navigates to the linked file.

**CSS for wikilinks** (add to `src/app.css`):
```css
.wikilink-decoration {
  color: var(--color-vigil-teal) !important;
  text-decoration: underline;
  text-decoration-style: dotted;
  cursor: pointer;
}
```

### Part 2: Backlinks Panel

**File:** `src/lib/components/editor/BacklinksPanel.svelte` (create new)

```svelte
<script>
  import { getBacklinks } from '$lib/tauri/commands';
  import { editorState } from '$lib/stores/editor';
  import { filesState } from '$lib/stores/files';

  let backlinks = $state<string[]>([]);
  let collapsed = $state(false);

  $effect(() => {
    const file = editorState.activeFile;
    const workspace = filesState.workspacePath;
    if (!file || !workspace || !file.endsWith('.md')) {
      backlinks = [];
      return;
    }
    getBacklinks(workspace, file).then((links) => {
      backlinks = links;
    });
  });
</script>

{#if backlinks.length > 0}
  <div class="backlinks-panel">
    <button class="panel-header" onclick={() => collapsed = !collapsed}>
      <span class="chevron" class:rotated={!collapsed}>▶</span>
      Backlinks ({backlinks.length})
    </button>
    {#if !collapsed}
      <ul class="backlink-list">
        {#each backlinks as link}
          <li>
            <button onclick={() => navigateToFile(link)}>
              {link}
            </button>
          </li>
        {/each}
      </ul>
    {/if}
  </div>
{/if}
```

**Placement:** Below the editor area, as a collapsible panel. Or as a
section in the sidebar — decide based on available space.

**Styling:**
```
border-t border-vigil-border
bg-vigil-surface/30
p-2
text-xs text-vigil-text-muted
max-h-40 overflow-y-auto
```

Backlink items:
